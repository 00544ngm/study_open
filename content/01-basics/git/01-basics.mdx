## 入门

```bash
# 配置身份
git config --global user.name "Your Name"
git config --global user.email "your@email.com"

# 克隆仓库
git clone https://github.com/username/project.git

# 基本工作流
git add .          # 暂存所有变更
git commit -m "feat: add agent router"  # 提交
git push           # 推送到远程
git pull           # 拉取最新代码
```

## 核心

### AI Agent 项目中的 Git 实践

```bash
# 分支开发
git checkout -b feature/rag-pipeline
git add src/rag/
git commit -m "feat: implement RAG pipeline"
git push -u origin feature/rag-pipeline

# 查看变更
git status         # 查看工作区状态
git log --oneline  # 简洁的提交历史
git diff           # 查看未暂存的变更

# 撤销操作
git restore file.py           # 撤销工作区修改
git restore --staged file.py  # 取消暂存
git reset --soft HEAD~1       # 撤销上次提交，保留更改
```

### 提交信息规范

AI Agent 项目推荐 Conventional Commits：

```
feat: 新功能
fix: 修复
docs: 文档
refactor: 重构
test: 测试
chore: 构建/工具
```
