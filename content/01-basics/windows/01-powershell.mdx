## 入门

```powershell
# 文件操作
Get-ChildItem          # ls / dir，列出文件
Set-Location Desktop   # cd，切换目录
Get-Content file.txt   # cat / type，查看文件
Remove-Item file.txt   # del / rm，删除文件
Copy-Item a.txt b.txt  # copy / cp，复制文件
Move-Item a.txt dir/   # move / mv，移动文件
New-Item -ItemType Directory dir  # mkdir，创建目录

# 查看内容
Get-Content log.txt -Tail 100     # tail -n 100
Get-Content log.txt -TotalCount 20 # head -20
Select-String "ERROR" log.txt     # grep
```

## 核心

```powershell
# 系统管理
Get-Process              # ps，查看进程
Stop-Process -Name chrome  # kill，终止进程
Get-Service              # 查看服务
Restart-Service docker   # 重启服务

# 网络
Test-Connection google.com -Count 1  # ping
Invoke-WebRequest -Uri https://api.openai.com  # curl
Get-NetTCPConnection     # netstat，查看端口

# 环境变量
$env:Path               # 查看 Path 变量
$env:OPENAI_API_KEY = "sk-..."  # 设置临时变量
[System.Environment]::SetEnvironmentVariable("OPENAI_API_KEY", "sk-...", "User")
```

## 进阶

### WSL 配置（Windows 下开发 AI Agent 推荐）

```powershell
# 安装 WSL
wsl --install
wsl --set-default-version 2

# 在 WSL 中开发
wsl -d Ubuntu
# 之后所有 Linux 命令在 WSL 终端中使用
```

### 开发环境配置

```powershell
# 使用 winget 安装开发工具
winget install Microsoft.PowerShell
winget install Git.Git
winget install Docker.DockerDesktop
winget install Microsoft.VisualStudioCode

# 配置 PATH
$env:Path += ";C:\tools"
```
