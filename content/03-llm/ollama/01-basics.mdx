## 安装

```bash
# macOS
brew install ollama

# Linux
curl -fsSL https://ollama.com/install.sh | sh

# Windows
# 从 https://ollama.com 下载安装包
```

## 下载模型

```bash
# 查看可用模型
ollama list

# 拉取模型
ollama pull llama3.2          # Meta Llama 3.2 (3B)
ollama pull qwen2.5           # 通义千问 2.5
ollama pull mistral           # Mistral
ollama pull qwen2.5-coder     # 代码专用

# 运行并交互
ollama run qwen2.5
>>> Python 的装饰器是什么？
```

## OpenAI 兼容 API

Ollama 启动后默认提供 OpenAI 兼容接口：

```bash
ollama serve
# 服务运行在 http://localhost:11434
```

```python
from openai import OpenAI

client = OpenAI(
    base_url="http://localhost:11434/v1",
    api_key="ollama",  # 任意值
)

response = client.chat.completions.create(
    model="qwen2.5",
    messages=[{"role": "user", "content": "你好"}],
)
print(response.choices[0].message.content)
```

## LangChain 集成

```python
from langchain_ollama import ChatOllama

llm = ChatOllama(
    model="qwen2.5",
    temperature=0.7,
)

response = llm.invoke("用 Python 实现一个快速排序")
print(response.content)
```

## 结构化输出

```python
from pydantic import BaseModel
from langchain_ollama import ChatOllama

class Code(BaseModel):
    language: str
    code: str
    explanation: str

llm = ChatOllama(model="qwen2.5")
structured = llm.with_structured_output(Code)

result = structured.invoke("用 Python 写一个二分查找")
print(result.language, result.code[:100], sep="\n")
```

## 管理模型

```bash
# 查看运行的模型
ollama ps

# 停止模型
ollama stop qwen2.5

# 删除模型
ollama rm qwen2.5

# 创建自定义模型
ollama create mymodel -f ./Modelfile
```

## Modelfile

类似 Dockerfile，自定义模型配置：

```dockerfile
FROM qwen2.5

# 设置系统提示词
SYSTEM """你是一个 Python 编程助手，回答简洁、代码优先。"""

# 调整参数
PARAMETER temperature 0.3
PARAMETER top_p 0.9
```

```bash
ollama create python-assistant -f ./Modelfile
ollama run python-assistant
```

| 命令 | 说明 |
|------|------|
| `ollama pull <model>` | 下载模型 |
| `ollama run <model>` | 运行交互 |
| `ollama serve` | 启动 API 服务 |
| `ollama list` | 列出本地模型 |
| `ollama ps` | 查看运行中的模型 |
