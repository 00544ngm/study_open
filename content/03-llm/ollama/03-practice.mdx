### 练习 3：自定义模型

```bash
# 1. 创建 Modelfile
cat > Modelfile << 'EOF'
FROM llama3.2

# 设置系统提示词
SYSTEM "You are a coding assistant specialized in Python."

# 设置参数
PARAMETER temperature 0.1
PARAMETER top_p 0.9
PARAMETER max_tokens 2048
EOF

# 2. 构建自定义模型
ollama create my-coder --modelfile Modelfile

# 3. 测试自定义模型
```
