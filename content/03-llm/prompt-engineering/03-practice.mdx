## 练习

### 目标

通过对比实验掌握 Prompt 设计技巧，体验不同 Prompt 结构对输出质量的影响。

### 练习 1：模糊 vs 精确 Prompt

```text
Task: 写一个 Python 函数

❌ 模糊版本：
"帮我把列表里的数字加起来"

✅ 精确版本：
"实现一个 Python 函数 sum_list(numbers: list[float]) -> float，
返回列表中所有数字的和。
如果列表为空，返回 0.0。
如果包含非数字元素，抛出 TypeError。"

实验步骤：
1. 将两个 Prompt 分别发给 LLM
2. 对比输出质量
3. 记录差异：

   模糊版本输出有什么问题？
   ________________________________

   精确版本有哪些改进？
   ________________________________
```
