## ReAct 模式

推理 + 行动循环，Agent 自主决定何时推理、何时行动：

```
观察(Observation) → 思考(Thought) → 行动(Action) → 观察 → ...
```

```python
from langchain_openai import ChatOpenAI
from langchain.agents import create_react_agent, AgentExecutor
from langchain.tools import tool

llm = ChatOpenAI(model="gpt-4o")

@tool
def search(query: str) -> str:
    """搜索互联网"""
    return f"搜索结果: {query}"

agent = create_react_agent(llm, [search], prompt)
executor = AgentExecutor(agent=agent, tools=[search])

result = executor.invoke({"input": "今天的 AI 新闻有什么？"})
```

## Plan-and-Execute

先规划再执行，适合复杂任务分解：

```python
class PlanAndExecute:
    def __init__(self, llm):
        self.llm = llm

    def plan(self, task: str) -> list[str]:
        prompt = f"将任务分解为步骤：{task}\n步骤："
        response = self.llm.invoke(prompt)
        return [s.strip() for s in response.content.split("\n") if s.strip()]

    def execute(self, steps: list[str]) -> list[str]:
        results = []
        for step in steps:
            result = self.llm.invoke(f"执行步骤：{step}")
            results.append(result.content)
        return results

    def run(self, task: str):
        steps = self.plan(task)
        results = self.execute(steps)
        return self.llm.invoke(
            f"汇总结果：\n" + "\n".join(results)
        )
```

## Tool-Use 模式

Agent 通过工具与外部世界交互：

```python
class ToolUseAgent:
    def __init__(self, llm, tools: dict):
        self.llm = llm
        self.tools = tools

    def run(self, user_input: str) -> str:
        response = self.llm.invoke(
            f"可用工具：{list(self.tools.keys())}\n"
            f"用户：{user_input}\n"
            f"如果需调用工具，返回 TOOL: tool_name | 参数"
        )

        if response.content.startswith("TOOL:"):
            tool_call = response.content[5:].strip()
            tool_name, args = tool_call.split("|", 1)
            result = self.tools[tool_name.strip()](args.strip())
            return self.llm.invoke(f"工具结果：{result}\n请回复用户")
        return response.content
```

## Reflection 模式

Agent 自我反思改进输出：

```python
class ReflectionAgent:
    def generate(self, task: str) -> str:
        return self.llm.invoke(f"完成任务：{task}")

    def reflect(self, task: str, output: str) -> str:
        return self.llm.invoke(
            f"任务：{task}\n输出：{output}\n"
            f"请评价质量并指出改进点："
        )

    def refine(self, task: str, output: str, feedback: str) -> str:
        return self.llm.invoke(
            f"任务：{task}\n原始输出：{output}\n反馈：{feedback}\n"
            f"请改进输出："
        )

    def run(self, task: str, max_iterations=3):
        output = self.generate(task)
        for _ in range(max_iterations):
            feedback = self.reflect(task, output)
            if "无需改进" in feedback:
                break
            output = self.refine(task, output, feedback)
        return output
```

| 模式 | 适用场景 |
|------|---------|
| ReAct | 通用问答、搜索 |
| Plan-and-Execute | 复杂任务分解 |
| Tool-Use | 需要外部数据 |
| Reflection | 高质量输出要求 |
