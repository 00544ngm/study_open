```text

1. 问题分类器（LLM 节点）

   - 输出：问题类别（技术/概念/操作）

2. 知识库检索（知识检索节点）
   - 根据问题类别选择不同知识库

3. 答案生成（LLM 节点）
   - 输入：原始问题 + 检索到的文档
   - 约束：基于检索内容回答，不要编造

4. 答案格式化（Code 节点）
   - 将答案格式化为 markdown

请画出节点连接关系（用文本描述）：

________________________________
________________________________
```
