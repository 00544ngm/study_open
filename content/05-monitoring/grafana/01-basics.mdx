## 入门

```bash
# Docker 启动
docker run -d -p 3000:3000 --name=grafana grafana/grafana

# 访问 http://localhost:3000
# 默认账号/密码: admin / admin
```

```yaml
# 使用 docker-compose
services:
  grafana:
    image: grafana/grafana:latest
    ports:
      - "3000:3000"
    volumes:
      - grafana-data:/var/lib/grafana
    environment:
      - GF_SECURITY_ADMIN_PASSWORD=admin123

volumes:
  grafana-data:
```

## 核心

### 添加 Prometheus 数据源

1. 打开 Configuration → Data Sources → Add data source
2. 选择 Prometheus
3. URL 填写 `http://prometheus:9090`
4. 点击 Save & Test

```yaml
# 或者通过配置文件预配
# provisioning/datasources/prometheus.yml
apiVersion: 1
datasources:
  - name: Prometheus
    type: prometheus
    url: http://prometheus:9090
    access: proxy
    isDefault: true
```
