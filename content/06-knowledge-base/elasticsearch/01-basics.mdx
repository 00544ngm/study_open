## 安装

```bash
# Docker
docker run -d --name es \
  -p 9200:9200 \
  -e "discovery.type=single-node" \
  -e "xpack.security.enabled=false" \
  docker.elastic.co/elasticsearch/elasticsearch:8.12.0

# 验证
curl http://localhost:9200
```

| 概念 | ES 类比 | 关系型数据库类比 |

## 搜索

```bash
# 全文搜索
GET /articles/_search
{
  "query": {
    "match": {
      "content": "搜索引擎"
    }
  }
}

# 精确匹配
GET /articles/_search
{
  "query": {
    "term": {
      "tags": "搜索"
    }
  }
}

# 组合查询
GET /articles/_search
{
  "query": {
    "bool": {
      "must": [{ "match": { "content": "搜索" } }],
      "filter": [{ "range": { "views": { "gte": 100 } } }]
    }
  }
}
```

## Python 客户端

```python
from elasticsearch import Elasticsearch

es = Elasticsearch("http://localhost:9200")

# 索引文档
doc = {
    "title": "RAG 技术实践",
    "content": "检索增强生成是一种结合搜索和生成的 AI 技术...",
    "tags": ["RAG", "AI", "搜索"],
}
resp = es.index(index="articles", document=doc)
print(resp["_id"])

# 搜索
resp = es.search(
    index="articles",
    query={
        "match": {"content": "RAG"}
    },
)
for hit in resp["hits"]["hits"]:
    print(hit["_source"]["title"], hit["_score"])
```

| 功能 | 说明 |
|------|------|
| match | 全文搜索 |
| term | 精确匹配 |
| bool | 组合查询 |
| range | 范围过滤 |
| fuzzy | 模糊搜索 |
