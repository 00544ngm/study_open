### 练习 3：多阶段构建

```dockerfile

# 阶段 1：构建阶段 - 安装编译依赖
# 阶段 2：运行阶段 - 仅复制必要的产物

# 阶段 1: builder
FROM python:3.12-slim AS builder
WORKDIR /build

# 复制依赖文件
COPY requirements.txt .

# 安装依赖到 /build/venv
RUN python -m venv /build/venv && \
    /build/venv/bin/pip install --no-cache-dir -r requirements.txt

# 复制源码
COPY ./app ./app
```
