### 练习 2：审查清单实操

用你学到的 AI 审查清单，审查以下 AI 生成的代码：

```python
def upload_file(file):
    file.save(f"/tmp/{file.filename}")
    return {"path": f"/tmp/{file.filename}"}

def delete_user(user_id):
    db.execute(f"DELETE FROM users WHERE id = {user_id}")
    return {"deleted": True}

def send_welcome(email):
    import subprocess
    subprocess.run(f"echo 'Welcome!' | mail -s 'Welcome' {email}", shell=True)
```

**逐项检查**：
- 安全性：有没有路径穿越？命令注入？
- 正确性：逻辑对不对？
- 健壮性：异常进来了会怎样？
