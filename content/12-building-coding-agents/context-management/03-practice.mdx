## 练习

### 目标

实现一个上下文窗口管理器，模拟 LLM 的上下文限制，并实现压缩和优先级策略。

**测试**：
```python
cw = ContextWindow(max_tokens=1000)

# 添加多条消息
for i in range(20):
    cw.add_message("user", f"这是第{i+1}条消息，包含一些示例文本。")

# 验证窗口没有超过限制
assert cw.estimate_tokens() <= cw.max_tokens
```
