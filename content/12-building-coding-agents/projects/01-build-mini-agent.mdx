## 项目：Mini Code Agent

### 目标

从零构建一个简化版的 Claude Code：通过 LLM API 调用，实现对文件的读取和编辑功能，并具备基本的命令执行能力和安全审批。

### 架构

```
用户输入 → Agent Loop → LLM API
                ↓
          ┌─────┴──────┐
          ↓             ↓
    工具层          安全审批层
   ├ FileOps       ├ Policy Engine
   ├ ShellExec     ├ Approval Flow
   ├ CodeSearch    └ Audit Log
   └ LSP Client
```

### 阶段 1：Agent 循环

```python
class Agent:
    def __init__(self, llm, tools: list[Tool], security: SecurityPolicy):
        self.llm = llm
        self.tools = {t.name: t for t in tools}
        self.security = security
        self.history = []

    async def run(self, user_input: str):
        self.history.append({"role": "user", "content": user_input})

        while True:
            response = await self.llm.chat(self.history)

            if response.tool_call:
                # 安全审批
                approved = self.security.check(response.tool_call)
                if approved:
                    result = await self.tools[response.tool_call.name].execute(
                        **response.tool_call.args
                    )
                    self.history.append({
                        "role": "tool",
                        "content": result
                    })
                else:
                    self.history.append({
                        "role": "tool",
                        "content": "Operation rejected by security policy"
                    })
            else:
                # LLM 返回最终回复
                return response.content
```

### 阶段 2：文件操作工具

```python
class ReadFile(Tool):
    name = "read_file"
    description = "读取文件的指定行范围"

    async def execute(self, path: str, start: int = 0, end: int | None = None):
        with open(path) as f:
            lines = f.readlines()
        return "".join(lines[start:end])

class EditFile(Tool):
    name = "edit_file"
    description = "行级文件编辑"

    async def execute(self, path: str, edits: list[dict]):
        # edits = [{line: 10, content: "new line"}, ...]
        with open(path) as f:
            lines = f.readlines()
        for edit in edits:
            lines[edit["line"]] = edit["content"] + "\n"
        with open(path, "w") as f:
            f.writelines(lines)
        return f"Edited {len(edits)} lines"
```

**任务**: 实现 `ReadFile` 和 `EditFile` 工具，包含备份和回滚机制。

### 阶段 3：命令执行工具

```python
class ShellExec(Tool):
    name = "execute_command"
    description = "执行 shell 命令"

    async def execute(self, command: str, timeout: int = 30):
        try:
            proc = await asyncio.create_subprocess_shell(
                command,
                stdout=asyncio.subprocess.PIPE,
                stderr=asyncio.subprocess.PIPE
            )
            stdout, stderr = await asyncio.wait_for(
                proc.communicate(), timeout=timeout
            )
            return stdout.decode() or stderr.decode()
        except asyncio.TimeoutError:
            proc.kill()
            return f"Command timed out after {timeout}s"
```

### 阶段 4：安全审批

```python
class SecurityPolicy:
    def check(self, tool_call) -> bool:
        if tool_call.name == "execute_command":
            return self._check_command(tool_call.args["command"])
        return True

    def _check_command(self, cmd: str) -> bool:
        dangerous = ["rm -rf /", "sudo", "> /dev/sda"]
        return not any(d in cmd for d in dangerous)
```
