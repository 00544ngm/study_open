## 一键部署脚本

将以下内容保存为 `deploy.bat`，**以管理员身份运行**：

```bat
@echo off
chcp 65001
echo ================================
echo DeepSeek 本地一键部署开始
echo ================================

:: 1. 创建目录
mkdir C:\ai\llama.cpp 2>nul
cd /d C:\ai

:: 2. 下载 llama.cpp 预编译包（CUDA 版）
echo 正在下载 llama.cpp（带 CUDA 加速）...
curl -L -o llama-cuda.zip https://github.com/ggerganov/llama.cpp/releases/latest/download/llama-b3928-bin-win-cuda-x64.zip
powershell -Command "Expand-Archive -Path llama-cuda.zip -DestinationPath llama.cpp -Force"
del llama-cuda.zip

:: 3. 创建模型目录
mkdir llama.cpp\models 2>nul

echo ================================
echo 部署完成！
echo ================================
echo 下一步：手动下载 GGUF 模型放到 C:\ai\llama.cpp\models\
echo 然后运行：llama-cli.exe -m models\模型文件名.gguf -ngl 999 -p "你好"
echo ================================
pause
```

## 手动安装

### 下载预编译包

推荐使用预编译 exe，解压直接用，**不需要自己编译**。

**带 CUDA 加速版（推荐，NVIDIA 显卡用）：**

```powershell
Invoke-WebRequest -Uri "https://github.com/ggerganov/llama.cpp/releases/latest/download/llama-b3928-bin-win-cuda-x64.zip" -OutFile "llama-cuda.zip"
Expand-Archive -Path llama-cuda.zip -DestinationPath C:\ai\llama.cpp -Force
```

**纯 CPU 版（无 NVIDIA 显卡用）：**

```powershell
Invoke-WebRequest -Uri "https://github.com/ggerganov/llama.cpp/releases/latest/download/llama-b3928-bin-win-no-cuda-x64.zip" -OutFile "llama-cpu.zip"
Expand-Archive -Path llama-cpu.zip -DestinationPath C:\ai\llama.cpp -Force
```

> **VC++ 编译环境：一般不需要**
>
> llama.cpp 提供**编译好的 exe**，直接下载解压就用，不用自己装 VS Build Tools。
>
> 解压后 `C:\ai\llama.cpp` 里就有 `llama-cli.exe` 和 `llama-server.exe`，直接运行即可。
>
> 只有想改源码或自定义编译参数时，才需要装 VS Build Tools + CMake。

### 关于编译

预编译 exe 解压直接用，**不需要自己编译**。只有想改源码或自定义编译参数时，才需要装 VS Build Tools + CMake：

```cmd
cd C:\ai\llama.cpp
cmake -B build -DGGML_CUDA=ON
cmake --build build --config Release
```

## 运行模型

```cmd
cd C:\ai\llama.cpp
llama-cli.exe ^
  -m models\DeepSeek-R1-Distill-Qwen-14B-Q4_K_M.gguf ^
  -ngl 999 ^
  -p "你好，请介绍一下你自己" ^
  -n 512
```

**参数说明：**
- `-m` — 指定模型文件路径
- `-ngl 999` — 将尽可能多的层卸载到 GPU（关键性能参数）
- `-p` — 输入提示词
- `-n` — 最大生成 token 数

## 启动 API 服务器

```cmd
cd C:\ai\llama.cpp
llama-server.exe ^
  -m models\DeepSeek-R1-Distill-Qwen-14B-Q4_K_M.gguf ^
  -ngl 999 ^
  --port 8080
```

访问 http://localhost:8080 即可查看 API 文档。
